Die Angaben sind anonym.

Dafür benutzen wir nicht deinen Namen, sondern einen „Code“.

Bitte bilde deinen Code folgendermaßen:

  • die ersten beiden Buchstaben deines Vornamens

  • den Tag deines Geburtsdatums

  • und die letzten beiden Ziffern deiner Telefonnummer.

    • Beispiel: MI0965 für Michael (Vorname), 09. Dezember 1993 (Geburtsdatum), 0176-XXXXXX 65 (Telefonnummer)

    Bitte trage deinen Code nach obigem Muster in das Textfeld ein.
