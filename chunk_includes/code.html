Die Angaben sind anonym.

Dafür benutzen wir nicht Deinen Namen, sondern einen „Code“.

Bitte bilde Deinen Code folgendermaßen:

  • die ersten beiden Buchstaben Deines Geburtsortes

  • den Tag des eigenen Geburtsdatums

  • und die ersten beiden Buchstaben des eigenen Vornamens.

    • Beispiel: KR08PE für Krefeld (Geburtsort), 08. Januar 1999 (Geburtsdatum), Peter (eigener Name)

    Bitte trage Deinen Code nach obigem Muster in das Textfeld ein.
